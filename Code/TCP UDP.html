1UDP connection

1.1Server

Socket Creation: Create a UDP socket using the socket() function

Socket Binding: Bind the socket to an address and port using the bind() function. This makes the server listen for incoming data on that port.

Receiving Data: Use the recvfrom() function to receive data sent by clients. The server can process the data and optionally send a response back.

Sending Data: Use the sendto() function to send data to the client's socket.

1.2Client

Socket Creation: Create a UDP socket using the socket() function.

Sending Data: Use the sendto() function to send data to the server's address and port.

Receiving Data: Optionally, the client can receive a response from the server using recvfrom() if the server sends one.

3TCP connection

3.1Server

Socket Creation: Create a TCP socket using the socket() function. [gives a fd]

Socket Binding: Bind the socket to an address and port using the bind() function. This allows the server to listen for incoming connections on that port. [gives a port adr and ip adr]

Listening: Set the socket to listen for incoming connections using the listen() function.

Accepting Connections: Use the accept() function to accept an incoming connection from a client. This creates a new socket for the established connection.

Sending and Receiving Data: Use the send() and recv() functions to send and receive data over the established connection.

3.2Client

Socket Creation: Create a TCP socket using the socket() function.

Connecting: Use the connect() function to establish a connection to the server's address and port.

Sending and Receiving Data: Use the send() and recv() functions to send and receive data over the established connection.
